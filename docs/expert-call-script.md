# Скрипт квалификационного звонка с экспертом
**Академия Личности | Версия 1.0 | 2026-05-08**

---

> **Длительность:** 30–40 минут
> **Цель:** понять — наш человек или нет. Не продать, не убедить. Квалифицировать.
> **Результат звонка:** один из трёх статусов → ДА / ПОКА НЕТ / НЕТ

---

## ПЕРЕД ЗВОНКОМ — 5 минут

- [ ] Посмотреть соцсети / сайт / TG-канал эксперта
- [ ] Выяснить: какой домен (тело / психика / питание / коучинг / другое)?
- [ ] Примерно оценить аудиторию (есть / нет / неясно)
- [ ] Проверить: есть ли ИП или самозанятость?
- [ ] Записать 1–2 конкретных наблюдения о нём — использовать в открытии

---

## БЛОК 1 — ОТКРЫТИЕ (2–3 мин)

**Цель:** человек понял зачем разговор, расслабился, говорит свободно

### Фаза 1 (сейчас — мы ищем):
> "Мы сейчас формируем экспертный костяк нового формата сообщества —
> не просто школы, а пространства, где человек с любым жизненным запросом
> получает маршрут через нескольких специалистов.
> Я посмотрел(а) твою работу и подумал(а), что ты можешь быть органичной
> частью этого. Хочу 30 минут поговорить — понять, резонирует ли концепция,
> и подходим ли мы друг другу."

### Фаза 2 (потом — к нам обращаются):
> "Рад(а), что ты написал(а). Прежде чем я расскажу подробности —
> хочу сначала понять тебя и твою практику. Это важнее,
> чем сразу показывать презентацию. Договорились?"

---

## БЛОК 2 — ИХ ИСТОРИЯ (7–10 мин)

**Цель:** понять, где человек сейчас и откуда пришёл

- [ ] **Сколько лет в практике?** Что было до этого?

- [ ] **Как сейчас выглядит работа?**
  → индивидуально / группы / курсы / всё вместе?
  → онлайн / офлайн / оба?

- [ ] **Есть ли своя аудитория?**
  → подписчики, сообщество, база клиентов?
  → примерный размер (не давить — спросить мягко)

- [ ] **Как оформлена деятельность?**
  → ИП / самозанятый / юрлицо / никак?

- [ ] **Выдаёт ли что-то клиентам / студентам документально?**
  → сертификат, диплом, справку?

---

## БЛОК 3 — ИХ БОЛИ КАК ПРАКТИКА (7–10 мин)

**Цель:** найти точки, где Academy решает их реальную проблему

- [ ] **Что сейчас самое тяжёлое в работе?**
  *(слушать внимательно — это главный блок)*

- [ ] **Бывает ли, что клиент приходит с запросом не по вашей теме?**
  → Куда направляете? Есть ли проверенные коллеги?

- [ ] **Как решаете юридику и бухгалтерию?**
  → договоры, оплата, налоги — это в радость или в тягость?

- [ ] **Если ведёте обучение — что происходит с документами студентов?**
  → выдаёте что-то официальное? Запрашивают ли?

- [ ] **Есть ли ощущение профессионального одиночества?**
  → коллеги, обмен случаями, поддержка?

### Сигналы — фиксировать (не озвучивать пока):

| Что услышали | Что это значит |
|-------------|----------------|
| "Устал от бумаг и договоров" | Боль с операционкой — наш оффер попадает |
| "Часто не знаю, куда отправить клиента" | Готов к системному маршруту |
| "Хочу чтобы студенты получали диплом" | Нужна лицензия ДПО |
| "Нет коллег рядом, варюсь в себе" | Ценит сообщество |
| "Аудитория есть, но не знаю что с ней делать" | Готов к структуре |

---

## БЛОК 4 — ИХ ВЗГЛЯД НА КЛИЕНТА И РАБОТУ (5–7 мин)

**Цель:** проверить холистическое мышление — ключевой критерий

- [ ] **Когда клиент приходит к вам — с чем чаще всего?**
  → и что за этим стоит на самом деле, под поверхностным запросом?

- [ ] **Бывает ли, что понимаете: человеку нужна работа в другом направлении параллельно?**
  → что делаете в такой момент?

- [ ] **Как смотрите на смежных специалистов?**
  → психолог + нутрициолог + телесник — это конкуренты или партнёры?

- [ ] **Верите ли, что у одного клиента несколько точек роста одновременно?**
  → или лучше решать по одной?

### Зелёные флаги ✅ (холистическое мышление есть):
- "Да, я часто вижу что человеку нужен ещё и [смежный специалист]"
- "Я сам(а) прохожу несколько практик одновременно"
- "Лучший результат — когда работаем командой"
- "Мой метод не решает всё, я честен(на) с клиентами"

### Красные флаги ❌ (стоп, не наш человек):
- "Моя система комплексная, мне другие не нужны"
- "Зачем делиться клиентами с коллегами?"
- "Психология не связана с телом / питанием / [чем угодно]"
- "Клиент должен работать только с одним специалистом"

---

## БЛОК 5 — ПРЕЗЕНТАЦИЯ КОНЦЕПЦИИ (5–7 мин)

**Цель:** показать идею маршрута и оффер — только после того, как их выслушали

> "Теперь хочу поделиться тем, что мы строим — и посмотреть, откликается ли.
> Мы не строим просто школу с дипломами. Мы строим портал:
> человек входит с любым жизненным запросом — усталость, потеря смысла,
> проблемы в теле или в отношениях — и мы проводим диагностику,
> а потом ведём его по маршруту через нескольких специалистов.
> Питание, психика, тело, коучинг, йога — каждый делает своё,
> и вместе это даёт человеку осознанность и реальный сдвиг.
> Часть клиентов потом хочет сами стать специалистами —
> и тогда они проходят обучение и получают диплом ДПО государственного образца.
> Это наша лицензия — бессрочная, всё официально."

**Пауза. Дать им отреагировать.**

Затем — оффер:
> "Мы сейчас формируем костяк — 8 специалистов по разным доменам.
> Три человека уже внутри. Что мы берём на себя: юридику, договоры,
> документооборот, дипломы, регистрацию в государственном реестре.
> Ты занимаешься только своей практикой — мы маршрутизируем к тебе клиентов
> и даём статус 'эксперт Академии Личности'.
> Для тех, кто входит сейчас — первые 3 месяца без комиссии,
> чтобы почувствовать поток и формат."

---

## БЛОК 6 — РЕАКЦИЯ И УТОЧНЕНИЯ (5 мин)

**Цель:** прочитать, насколько человек внутри концепции

- [ ] **Что из этого откликается больше всего?**

- [ ] **Что вызывает вопросы или сомнения?**

- [ ] **Есть ли уже опыт работы в партнёрстве или команде специалистов?**

- [ ] **Как сейчас смотришь на сентябрь 2026?**
  *(если есть актуальность закона о психологах — упомянуть)*
  > "С 1 сентября 2026 работа без ДПО становится вне закона для ряда практик.
  > Наша лицензия — один из немногих способов это решить быстро и официально."

---

## БЛОК 7 — СЛЕДУЮЩИЙ ШАГ (3–5 мин)

**Только если видите зелёные флаги:**
> "По ощущениям — вижу, что концепция резонирует.
> Предлагаю следующий шаг: я пришлю тебе описание формата участия,
> посмотри спокойно. Через 3–4 дня созвонимся ещё раз — уже предметно,
> с конкретными условиями. Как тебе?"

**Если сомневаетесь:**
> "Понял(а). Давай я пришлю материалы — там всё подробно.
> Посмотри, и если откликнется — напиши. Мы не торопим."

**Если красные флаги:**
> "Спасибо за открытый разговор. Думаю, сейчас наши форматы
> немного в разных плоскостях. Буду рад(а) вернуться к теме,
> если что-то изменится — или просто остаться в профессиональном контакте."

## ИТОГОВЫЙ СКОРИНГ — заполнить сразу после звонка

| Критерий | 0 — нет | 1 — частично | 2 — да |
|----------|---------|-------------|--------|
| Домен нужный (закрывает незаполненную нишу) | | | |
| Активная практика (не только теория) | | | |
| Холистическое мышление (готов передавать клиентов) | | | |
| Боль с операционкой (юридика, документы) | | | |
| Ценит сообщество коллег | | | |
| ДПО-статус нужен или приветствуется | | | |
| Умеет объяснять просто (проверить по разговору) | | | |
| Реакция на концепцию (загорелся / равнодушен) | | | |

**Максимум: 16 баллов**

| Баллы | Решение | Действие |
|-------|---------|----------|
| 12–16 | ДА | Присылаем условия, назначаем предметный звонок |
| 8–11 | ПОКА НЕТ | Присылаем материалы, возвращаемся через 2–3 мес. |
| 0–7 | НЕТ | Вежливо закрываем, остаёмся в контакте |

## БЫСТРЫЙ СПРАВОЧНИК: зелёные и красные флаги

### ✅ Берём — если слышим:
- "Часто не знаю, куда направить клиента с другим запросом"
- "Устал(а) от юридики и договоров"
- "Хочу, чтобы мои студенты получали официальный документ"
- "Одиноко работать — хочется коллег рядом"
- "Лучший результат у клиента — когда несколько специалистов"
- "Интересно, как это работает в системе" *(задаёт вопросы о других экспертах)*

### ❌ Не берём — если слышим:
- "Мой метод закрывает всё, мне другие не нужны"
- "Зачем делиться клиентами?"
- "Я работаю один(а), так лучше контролировать качество"
- Не практикует — только преподаёт теорию
- Не готов соблюдать общие форматы (диагностика, передача клиентов)
- Хочет диплом, но не хочет быть частью сообщества
