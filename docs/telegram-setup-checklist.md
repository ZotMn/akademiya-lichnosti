## КАНАЛ «АКАДЕМИЯ РАЗВИТИЯ ЛИЧНОСТИ»

### Шаг 1 — Обновить описание
Открыть канал → **Редактировать** → вкладка **Описание** → вставить:

```
Академия Развития Личности — лицензированная ДПО-площадка для экспертов и студентов.

→ Курсы с государственным дипломом
→ Эфиры и практики экспертов сообщества
→ Нетворкинг и коллаборации

Запустить курс или найти специалиста: @Julia_Tsezar
```

### Шаг 2 — Почистить канал
Удалить все посты по списку ниже. Делать вручную или попросить Юлю.

**Удаляем однозначно:**
- [ ] Всё про Роспотребнадзор, проверки, лицензирование
- [ ] Репосты чужих материалов без контекста
- [ ] Опросы (устарели)
- [ ] Анонсы прошедших мероприятий
- [ ] Советы по инфобизнесу и прямым эфирам
- [ ] Конференционные фото без пояснений

**Смотрим и решаем:**
- Посты о Юле — можно оставить если актуальны, иначе удалить и переписать
- Посты об экспертах Академии — оставить если формат нормальный

### Шаг 3 — Опубликовать 5 якорных постов
По одному в день с 26 по 30 мая (тексты в файле `telegram-content-launch.md`).
После публикации поста 1 — **закрепить** его в канале.

## ГРУППА → «АКАДЕМИЯ РАЗВИТИЯ ЛИЧНОСТИ | СООБЩЕСТВО»

### Шаг 1 — Переименовать и обновить описание
Открыть группу → **Редактировать** → изменить:

**Название:**
```
Академия Развития Личности | Сообщество
```

**Описание:**
```
Живое сообщество экспертов и их учеников.

Здесь: эфиры, практики, нетворкинг, ответы на вопросы.

Каждый эксперт — в своей теме. Находи своего специалиста и общайся напрямую.

📌 Вход свободный
```

### Шаг 2 — Включить Forum mode (Темы)
Настройки группы → **Тип группы** → найти переключатель **«Форум»** → включить.

> На телефоне: три точки → Управление группой → Тип группы → Форум.
> Требует супергруппа (обычно уже является таковой при 200+ участниках или при ручном переводе).
> Если опция не видна — нужно сначала перевести в супергруппу: Настройки → Превратить в супергруппу.

### Шаг 3 — Создать базовые темы
После включения форума создаём 4 темы (кнопка «Новая тема» в интерфейсе):

| Тема | Описание темы |
|------|---------------|
| 📌 Добро пожаловать | Правила, навигация, старт |
| 📢 Анонсы и события | Все предстоящие эфиры |
| ✨ Юля Цезарь — Методология РОМ | Авторская методология, обучение |
| 💬 Общий чат | Знакомства и вопросы |

> Остальные темы (для каждого эксперта) добавляем по мере подключения — не заранее.

### Шаг 4 — Закрепить приветственный пост
В теме «📌 Добро пожаловать» опубликовать приветственный пост (текст в `telegram-content-launch.md`) → **закрепить**.
