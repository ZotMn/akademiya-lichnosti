# CLAUDE.md — Академия Личности

## Карта проекта

```
AkademiyaLichnosti/
├── CLAUDE.md          ← ты здесь: карта + правила
├── index.md           ← навигация + статус проекта
├── ROADMAP.md         ← план развития + чеклист задач
├── docs/
│   ├── product-brief.md      ← концепция, позиционирование, целевая аудитория
│   ├── market-research.md    ← исследование рынка
│   ├── meeting-summary.md    ← саммари встреч и ключевые решения
│   ├── dashboard.md          ← дашборд исследования (данные, метрики)
│   ├── site-brief.md         ← бриф сайта / описание текущего сайта
│   ├── entry-plan.md         ← план входа в управление
│   └── changelog.md          ← история всех изменений
├── assets/            ← изображения, схемы
└── versions/          ← резервные копии документов
```

## Правила работы

1. Перед работой читать `index.md` → открывать только нужные `docs/`
2. После каждой сессии обновлять: `ROADMAP.md`, `docs/changelog.md`, `index.md`
3. Все решения фиксировать в `docs/meeting-summary.md`
4. Данные исследования — в `docs/market-research.md` и `docs/dashboard.md`
